## 📁 Backend Development

### 1. Event Management System
Clone the repository from GitHub to your local machine.
Open the project folder and install the required dependencies using pip install django djangorestframework.
Run the Django server with python manage.py runserver from the folder containing manage.py.
Open http://127.0.0.1:8000/api/events/ in your browser to access the API locally.

### 1.Java Gradle Project
To run this project:
1. Clone or download the repository to your local system
2. Open the project folder in an IDE (IntelliJ / VS Code)
3. Run the project using Gradle (gradle build and gradle run) or execute the main Java file from src/main/java
### 2. Web Server using Docker
To run this project:
1. Clone or download the repository to your local system
2. Make sure Docker is installed and running
3. Open terminal in the project folder and run: docker build -t webserver .
4. Then run: docker run -p 8080:80 webserver and open http://localhost:8080 in browser.
